# API Automation Framework

## Overview
This project is an API automation for all HTTP Methods built using Java, Rest Assured, TestNG, Maven, Jackson,jayway. 
It automates REST API testing with maintainable and reusable code.

## Tech Stack
- Java
- Rest Assured
- TestNG
- Maven
- Jackson
- Lombok
- Jayway
- POJO

## Project Structure

src
├── main
│   └── java
├── test
│   └── java
├── resources
│   ├── testdata
pom.xml

## Features
- GET API automation
- POST API automation
- PUT API automation
- PATCH API Automation
- DELETE API automation
- POJO serialization and deserialization
- JSON Schema Validation
- TestNG support
- Maven project structure

## Prerequisites
- Java 11 or later
- Maven
- Eclipse or IntelliJ IDEA

## Installation

Clone the repository:

```bash
git clone https://github.com/Sakshi-Thakur-cpu/API-Automation-All-HTTP-Methods
```

Navigate to the project:

```bash
cd <repository-name>
```

Install dependencies:

```bash
mvn clean install
```

## Run the Tests

```bash
mvn test
```

Or run `testng.xml` from your IDE.
